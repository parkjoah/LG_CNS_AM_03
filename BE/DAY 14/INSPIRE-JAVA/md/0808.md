### Spring MVC - Framework 구조 특징

- 여러 계층(Controller, Service, Repository 등)으로 나뉘어 각자 역할을 수행
- 각 계층의 클래스들은 서로 의존 관계를 가질 수밖에 없음

- 설계 목표:

  - 응집도(Cohesion) ↑ : 각 클래스가 자신의 책임과 기능에 집중
  - 결합도(Coupling) ↓ : 클래스 간 의존성을 최소화해 변경에 유연하게 대응
